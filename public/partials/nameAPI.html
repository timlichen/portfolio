Names API - MEAN stack

This application is created using the MEAN stack and lets users add data to a Mongo database through the Mongoose framework. The data is displayed using the AngularJS framework.

This application utilizes the ExpressJS framework on top of a NodeJS server. This application began using just ExpressJS, NodeJS, and MongoDB. Adding and removing data was originally done using URL requests. Adding AngularJS to the application was a good way to connect all the dots in the MEAN technology stack.

This application also allows users to update the JSON data that has already been written to the database.
